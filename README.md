**DSLab Embedding &amp; Word2vec Study Repository**

  - [CBOW negative sampling](word2vec_CBOW_NS.ipynb): 송재용

  - 발표준비: 김율

- 6주차(5.3~5.13): 발표준비(기본 임베딩 설명, 오토인코더, 기타모델)
